## Para Compilar
```
$ gcc -c texture.c
$ gcc sistema_solar.c -o sistema_solar texture.o -lm -lglut -lGL -lGLU && ./sistema_solar
```

ou use os comandos

```
$ make
$ ./sistema_solar
```
